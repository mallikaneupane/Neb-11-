Introduction

This syllabus covers the essential topics in Computer Science for Grade 11 students under the NEB curriculum. It aims to provide a solid foundation in computer science principles and practices, preparing students for further studies in technology and related fields.

Chapters

Chapter 1: Computer System

This chapter introduces the fundamentals of computer systems, including:

  • Definition and Components: Understanding what a computer system is and its main components (hardware and software).
  • Types of Computers: Overview of different types of computers (personal computers, servers, mainframes) and their applications.
  • Input and Output Devices: Discussion on various input (keyboard, mouse) and output devices (monitor, printer).
  • Storage Devices: Understanding primary (RAM) and secondary storage (hard drives, SSDs) and their functions.
  • Computer Architecture: Basic concepts of computer architecture, including CPU, memory hierarchy, and data flow.

Chapter 2: Number System and Conversion, Boolean Logic

This chapter covers essential concepts of number systems and Boolean logic, including:

  • Number Systems: Understanding different number systems such as binary, decimal, octal, and hexadecimal.
  • Conversions: Techniques for converting numbers between different bases (e.g., binary to decimal, decimal to binary).
  • Boolean Logic: Introduction to Boolean algebra, including operations such as AND, OR, NOT, and their truth tables.
  • Applications: Understanding how number systems and Boolean logic are used in computer programming and digital circuits.

Chapter 3: Computer Software and Operating Systems

This chapter discusses the types of software and operating systems, including:

  • Types of Software: Overview of system software, application software, and utility software.
  • Operating Systems: Understanding the role of operating systems in managing hardware and software resources.
  • Popular Operating Systems: Discussion on various operating systems such as Windows, macOS, Linux, and their features.

Chapter 4: Application Package (Word, Excel, PowerPoint)

This chapter introduces common application software packages, including:

  • Word Processing: Overview of Microsoft Word, including document creation, formatting, and editing features.
  • Spreadsheets: Introduction to Microsoft Excel, covering basic functions, formulas, and data analysis tools.
  • Presentations: Understanding Microsoft PowerPoint, including slide creation, design, and presentation techniques.
  • Applications in Real Life: Discussion on how these applications are used in various fields such as business, education, and personal productivity.

Chapter 5: Programming Concepts and Logics (C Language)

This chapter covers fundamental programming concepts using the C language, including:

  • Programming Languages: Introduction to high-level programming languages, focusing on C.
  • Basic Syntax: Understanding the syntax and structure of C programming.
  • Data Types and Variables: Overview of different data types in C and how to declare and use variables.
  • Control Structures: Learning about conditional statements (if, else) and loops (for, while).
  • Functions: Introduction to functions in C, their definition, and how to use them.
  • Debugging: Basic techniques for debugging and troubleshooting C code.

Chapter 6: Web Technology-I (HTML + CSS)

This chapter introduces the basics of web technology, including:

  • Web Development: Understanding the difference between front-end and back-end development.
  • HTML: Introduction to HyperText Markup Language (HTML) and its role in creating web pages.
  • CSS: Overview of Cascading Style Sheets (CSS) for styling web pages.
  • Basic Web Page Structure: Understanding the structure of a simple HTML document and how to apply CSS styles.
  • Web Hosting: Basics of how websites are hosted and the importance of domain names.

Chapter 7: Multimedia

This chapter covers the concepts of multimedia, including:

  • Definition: Understanding what multimedia is and its components (text, images, audio, video).
  • Multimedia Applications: Overview of various applications of multimedia in education, entertainment, and advertising.
  • Audio and Video Formats: Introduction to different audio and video file formats and their uses.
  • Multimedia Tools: Overview of software tools used for creating and editing multimedia content (e.g., Adobe Photoshop, Audacity).
  • Interactive Multimedia: Understanding the concept of interactive multimedia and its applications in web design and gaming.

Chapter 8: Information Security and Cyber Law

This chapter introduces the legal aspects of information technology, including:

  • Information Security: Understanding the importance of information security and basic security measures.
  • Cyber Law: Overview of the laws governing the use of the internet and digital communication.
  • Intellectual Property Rights: Discussion on copyright, trademarks, and patents in the digital age.
  • Data Protection: Understanding data protection laws and regulations (e.g., GDPR) and their importance.
  • Cyber Crimes: Discussion on various types of cyber crimes (hacking, phishing, identity theft) and their legal implications.
  • Ethical Issues: Understanding the ethical considerations in information technology and the responsibilities of IT professionals.
